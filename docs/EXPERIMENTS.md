# Experiments Summary

## Benchmark
A benchmark of 200 diverse, labeled routing tasks spanning eight agent specialties: market and data analysis, web research, content generation, email and outreach, code and infrastructure automation, strategic decision-making, multi-agent orchestration, and risk/intelligence synthesis.

## Results Table

| Experiment | Decisions | Key Result |
|---|---:|---|
| Plateau Discovery | 4,000 | Cardiac routing plateaued at 92.5% with 100% cardiac accuracy |
| Agent Failure Recovery | 2,000 | Immediate failure dip (81.5%) followed by recovery to 96.0% and zero disabled-agent misroutes |
| Marker Transplantation | 3,000 | 84.5% immediate cardiac routing with transplanted markers vs 0.0% cold-start |
| Adversarial Robustness | 2,050 | 4.8% poison concentration with no overall-accuracy degradation in this run |
| Cross-Domain Transfer | 1,480 | Early transfer signal (14.6% vs 0.0% pass-1) but weaker late performance than in-domain cold start |

Detailed JSON outputs belong in `experiments/results/`.
